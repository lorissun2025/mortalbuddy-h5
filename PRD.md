# MortalBuddy H5 - 产品设计

## 技术方案
- 纯前端 H5，单页应用
- HTML + CSS + JavaScript（无框架，零依赖）
- localStorage 持久化
- 移动端优先，适配桌面
- 部署 Vercel

## 页面结构
- 单页面，底部 Tab 切换：
  1. 灵宠（首页）- 显示灵宠、修炼按钮
  2. 修炼 - 修炼/冥想/历练/闭关
  3. 背包 - 丹药/法宝/灵石
  4. 比武 - 对战/排行榜

## MVP 核心功能（第一版）
1. 孵化灵宠（随机物种+稀有度+属性）
2. 显示灵宠（CSS 绘制，非 ASCII）
3. 修炼（消耗灵石，获得经验，升级）
4. 突破（境界提升，有概率失败）
5. 喂食（增加灵力）
6. 历练（随机事件，文字冒险）
7. 背包系统（灵石、丹药）
8. 状态持久化（localStorage）

## 18 种灵宠视觉设计
用 CSS/emoji 混合，每种灵宠有独特的视觉效果：
- 基于稀有度显示不同光效（普通=无，优秀=蓝光，稀有=紫光，史诗=金光，传说=彩虹）
- 闪光版本有闪烁动画
