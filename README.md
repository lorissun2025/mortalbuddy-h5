# MortalBuddy H5

凡人修仙传 — 灵宠修仙养成 H5 小游戏。

## 玩法

- 孵化灵宠（18 种物种，1-5 星稀有度，1% 闪光概率）
- 修炼、喂食、抚摸、对话、历练、闭关
- 突破境界（练气 → 筑基 → 金丹 → 元婴 → 化神 → 炼虚 → 大乘 → 渡劫 → 真仙 → 仙尊）
- 天气系统影响修炼效率
- 丹药背包系统
- 数据通过 localStorage 本地持久化

## 技术栈

- 纯前端：HTML + CSS + JavaScript
- 零依赖，无框架
- 移动端优先，适配桌面
- 部署到 Vercel（静态站点）

## 本地运行

直接用浏览器打开 `index.html` 即可，无需任何构建步骤。

## 部署

```bash
vercel --prod
```

## 文件说明

| 文件 | 说明 |
|------|------|
| `index.html` | 主页面（HTML + CSS + UI 逻辑） |
| `game.js` | 游戏核心引擎（MortalBuddy 类） |
| `vercel.json` | Vercel 静态部署配置 |
| `PRD.md` | 产品设计文档 |
